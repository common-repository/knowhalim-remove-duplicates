== Description ==

KH Remove Duplicates is an essential plugin for those experiencing duplicate post issues. Installing the plugin is fast and easy, and with a few clicks of the mouse you can quickly and easily delete unwanted duplicate posts. This plugin also offers support for custom post types, giving you complete control over the content management process. With KH Remove Duplicates, you can rest easy knowing that these issues are in the past.

Here is a summary of what you can do:

*   You can select custom post types to remove duplicates
*   You can removes posts based on recent or oldest
*   It will send you an email once completed (you need to configure the email)
*   Quick and easy way to just remove duplicates.
*.  And finally.. it's free!

If you need any assistance, feel free to contact me at [KnowHalim](https://knowhalim.com/contact-me "Contact Halim")

== Installation ==

To install, you just need to follow these steps

1. Upload `kh-remove-duplicates.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to settings > Remove Duplicate Posts
